4. Setup Environment

   c. open above agent folder into cursor IDE
   d. install uv from https://docs.astral.sh/uv/getting-started/installation/
      i. uv is an extremely fast Python package and project manager, written in Rust
   e. open terminal and staying at agent folder execute "uv sync"
      i. It will create a folder .venv and install python and all dependent package
   f. Create key in open ai at https://platform.openai.com/docs/quickstart
   g. Setup done
